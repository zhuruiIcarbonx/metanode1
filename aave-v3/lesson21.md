理解 Aave V3：供应和借贷的核心概念
Aave V3 是一个去中心化的流动性协议，用户可以作为供应方（贷款方）或借款方参与。本课程将探讨如何在主以太坊市场上与 Aave V3“核心实例”进行交互，重点介绍资产的供应和提取的基本操作，以及支撑这些操作的金融机制。

Aave V3 的核心在于构建一个数字资产借贷市场。向该协议提供代币的用户，让其他人可以借入这些代币。作为提供流动性的回报，代币供应商可以获得利息。这些利息来自借款人，他们为借入的资产支付利息。

关键概念：

供给（借贷）：用户将其加密资产存入 Aave V3 流动性池。这些供给的资产会随着时间的推移产生利息。

借贷：用户可以从 Aave V3 池中借入资产，前提是他们提供足够的抵押品。借款人需要支付贷款利息。

抵押品：用户提供的用于担保其借贷的资产。如果用户提供 DAI 并将其作为抵押品，他们就可以用 DAI 借入其他资产，例如 ETH。

利息动态：供应商赚取的利息直接由协议内借款人支付的利息提供资金。

如何向 Aave V3 提供资产：分步指南（DAI 示例）
向 Aave V3 提供资产可以让您获得收益。本演示将演示如何提供 DAI。

导航到“要提供的资产”：在 Aave V3 界面中，找到列出您可以提供的资产的部分。

选择资产和金额：在列表中选择 DAI。演示从钱包余额 1.63 DAI 开始。用户选择提供“最大”可用金额。

初始状态：本次补给交易前，用户“您的补给”栏目显示DAI为3.52，健康值1.49。

查看供应详情：出现一个标题为“供应 DAI”的弹出窗口。

金额：确认要供应的 1.6305... DAI。

交易概览：显示关键信息：

供应 APY：供应 DAI 的预期年收益率（例如 2.57%）。

抵押：表示资产是否可以用作抵押品（DAI 为“已启用”）。

健康因素影响：显示健康因素的预计变化（例如，从 1.49 到 2.19）。

执行补给：点击“补给DAI”按钮。

钱包确认：在已连接的钱包（例如 MetaMask）中确认交易。这将产生 Gas 费用。

交易确认：等待链上确认。系统将出现类似“全部完成。您已提供 1.63 DAI”的消息。

验证更新后的余额：

“您的供应”：此部分中的 DAI 余额将更新以反映新的总额（例如，5.15 DAI，即初始 3.52 DAI + 1.63 DAI 供应）。

健康系数：健康系数将更新至新的更高值（例如 2.19）。提供资产（尤其是用作抵押品的资产）通常会提升您的健康系数。

如何从 Aave V3 中提取资产：管理你的供应（DAI 示例）
用户可以提取其提供的资产，但如果资产被用作活跃贷款的抵押品，则可提取的金额可能会受到限制。

导航到“您的供应”：找到您想要提取的资产（例如，之前提供的 5.15 DAI）。

发起提款：单击提供的 DAI 旁边的“提款”按钮。

指定提款金额：出现“提取 DAI”弹出窗口。

点击“MAX”可显示您可以提取的最大金额。在本例中，它大约是 2.776 DAI，而不是完整的 5.15 DAI。

限制原因：用户拥有一笔活跃的 ETH 借款（例如，0.0009929 ETH，价值 1.81 美元，可在“您的借款”中查看）。提供的 DAI 将作为该借款的抵押品。提取全部 DAI 将导致支持该借款的抵押品减少，直至健康系数降至 1.0 以下，从而触发潜在的清算。

弹出窗口显示，如果提取约 2.77 DAI，健康系数将从 2.19 变为 1.00（或交易后为 1.01）。

审核并确认提款：

用户继续提取约 2.77 DAI 金额。

确认与健康因素影响相关的任何风险警告。

钱包确认：在您的钱包中确认交易（例如，MetaMask）。

交易确认：链上确认后，出现“全部完成，您提取了 2.78 DAI”之类的消息。

验证更新后的余额：

“您的供应”： DAI 余额更新为新的较低金额（例如 2.37 DAI）。

健康系数：健康系数更新为新的较低值（例如 1.01）。这非常接近清算阈值。

健康因素：Aave V3 中的风险管理
健康因素 (HF) 是 Aave V3 中的一个关键指标，代表您的贷款状况相对于抵押品的安全性。

HF > 1.0：您的头寸被视为安全且超额抵押。

HF ≤ 1.0：您的仓位抵押不足，面临被清算的风险。如果 HF 降至 1.0 或以下，其他用户（清算人）可以偿还您的部分债务，并索要相应金额的抵押品，外加罚金。

影响健康的因素：

增加HF：

提供更多资产（尤其是用作抵押品）。

偿还借入的资产。

如果您提供的抵押品的价值相对于您的债务大幅增加。

降低HF：

借入资产。

撤回抵押品。

如果您提供的抵押品的价值大幅下降，或者您借入的资产的价值大幅增加。

当您有活跃借款时，密切监控您的健康因素以避免清算至关重要。

解码 Aave V3 中的资产详细信息：全面了解（DAI 示例）
通过点击“待供应资产”部分中资产旁边的三个点并选择“详情”，用户可以访问丰富的信息。对于 DAI，这些详情包括：

储备规模：该特定代币在该特定市场的所有用户中提供给 Aave V3 协议的总价值（例如，DAI 为 1.6037 亿美元）。

可用流动性：池中当前可供用户借用的代币数量（例如，DAI 为 4491 万美元）。

利用率：当前借入的代币占总供应代币的百分比。其计算方式为：（总借入量 / 总供应量）或（储备规模 - 可用流动性）/储备规模（例如，DAI 的利用率是 72.00%）。该指标对利率有显著影响。

了解供应 APY 与 APR
供应 APY（年化百分比收益率）：供应方获得的年化利率，包含复利效应（例如，DAI 为 2.57%）。APY 反映的是定期将利息再投资或复利后的实际回报。

供应商年利率 (APR)：供应商赚取的年化利率，不考虑复利（例如，图表可能显示平均值为 2.14%，而 DAI 的最新点为 2.56%）。如果利息是复利，APY 通常会略高于 APR。

抵押品参数
这些参数定义了如何将资产用作抵押品：

可以作为抵押品：一个切换开关或指示器，显示所提供的资产是否可用于支持贷款（DAI 可以）。

最高贷款价值比 (LTV)：您可以借入的抵押品价值的最高百分比。例如，如果 DAI 的最高贷款价值比为 63%，那么提供价值 100 美元的 DAI 可以让您借入最多价值 63 美元的其他资产。

清算门槛：您的贷款头寸达到清算门槛时，其 LTV 百分比将达到清算门槛。如果您当前的 LTV（债务总额/抵押品总额）超过此门槛（例如，DAI 为 77%），您的健康系数将等于或低于 1.0，此时可能会发生清算。

清算罚金：根据被清算的抵押品金额额外收取的费用（例如，DAI 为 5.00%）。该罚金将作为奖励支付给清算人。如果您的 100 美元抵押品被清算以偿还债务，则将额外收取价值 5 美元的抵押品作为罚金。

利率模型
Aave V3 采用一种利率模型，通常以图表形式显示，显示借款 APR（变量）如何根据资产的利用率而变化。

总体趋势：随着利用率的增加（意味着借入更多的供应资产），借款年利率也趋于增加。

最佳利用率：该模型通常具有“最佳利用率”（例如，DAI 为 92%）。在此点，利率曲线的斜率通常会显著增加。该协议旨在将利用率保持在此最佳点附近或以下。如果利用率显著高于此点，借款利率将大幅上升，以激励还款并吸引新的供应，从而稳定流动性。

还简要提到了借款信息，其中详细说明了借入资产的参数，例如借款 APR，但通常在专门的借款课程中涵盖更广泛的内容。

实现有效导航的关键 Aave V3 UI 元素
熟悉 Aave V3 界面是管理您的头寸的关键：

顶部栏：显示基本概览信息，例如您的总体净值、净 APY、当前健康因素、风险详情、查看交易以及 Bridge GHO、Switch Tokens、Connected Wallet 和设置的选项。

您的资产：此部分详细说明您当前向协议提供的资产，显示其余额、您从中赚取的年化收益以及其作为抵押品的状态。它包含“切换”（启用/禁用资产作为抵押品）和“提现”按钮。

您的借款：此部分列出了您当前借入的所有资产、未偿还债务金额、您支付的年利率以及借款电力使用情况。它包含“切换”（例如，在适用的情况下在稳定借款利率和浮动借款利率之间切换）和“偿还”按钮。

待供应资产：您可以向 Aave 协议供应的代币的完整列表。对于每种资产，它会显示您的钱包余额、当前供应年利率以及是否可以用作抵押品。每种资产都提供“供应”按钮。

可借资产：可从协议借入的代币列表。它显示池中可用的代币数量以及当前的借入年利率/利率。每个资产都配有“借入”按钮。

资产详情页面：如前所述，此页面提供 Aave 市场内单个资产的深入信息和具体参数。

使用 Aave V3 的最佳实践
通过供应赚钱：供应代币是一种可能从您的加密货币持有中赚取被动利息的方式。

了解借贷成本：借贷会产生利息。请注意当前的借贷利率（APY/APR）及其如何根据市场利用率波动。

密切关注健康系数：这是管理借入仓位最关键的环节。较低的健康系数（接近1.0）会使您的抵押品面临较高的清算风险。

行动对健康指数的影响：请记住，提取抵押品或借入更多资产会降低您的健康指数。相反，提供更多抵押品或偿还债务则会提高您的健康指数。

提款限制：请注意，您提取所提供抵押品的能力可能会受到您的主动借款的限制，以维持健康的健康因素。

APY 与 APR： APY 包含复利效应，而 APR 不包含。APY 通常能更准确地反映复利作用下的潜在收益或成本随时间的变化。

清算风险：如果您的健康指数过低，清算将是一个真正的风险。清算将导致您损失部分抵押品，并额外支付清算罚金。请务必预留缓冲资金以避免这种情况。

了解 DeFi 借贷协议中的利用率
去中心化金融 (DeFi) 借贷协议彻底改变了用户借入和借出数字资产的方式。这些协议的核心机制是影响利率并反映市场动态的关键指标：利用率。本课程将深入讲解利用率的概念、计算方法以及它在 DeFi 生态系统中的重要性。

定义利用率

利用率是一个数值，通常表示为 0 到 1 之间的小数（或 0% 到 100% 之间的百分比）。在 DeFi 借贷协议中，利用率的主要作用是作为确定特定代币借贷利率的关键输入。本质上，它衡量的是借入某种资产的需求相对于协议内该资产的总供应量。利用率越高，通常表明借入该特定代币的需求越高。

利用率公式

利用率的计算很简单，由以下公式给出：

U = B / (S + B)

在哪里：

U表示利用率：正在计算的比率。

B代表总债务：这是用户当前借入的特定代币的总金额。至关重要的是，这个数字不仅包括借入的本金，还包括借款人有义务偿还的所有应计利息。

S表示可用流动性：这是当前存在于协议池中且可供用户借用的代币数量。

公式如何运作

该公式清晰地描绘了借贷协议（例如 Aave V3）中代币的市场状况。

分子B（总债务）代表主动借出且目前在借款人手中的代币部分。

分母S + B（可用流动性 + 总债务）表示协议管理的特定代币总量。它是池中剩余可用资金（S）与已借出资金（B）的总和。

因此，利用率U实际上告诉我们，协议持有的特定资产代币总量中，目前有多少比例被借用。

解释利用率值

利用率的值可以直接洞察贷款池的状态：

情况 1：所有代币均被借出（U = 1）

如果为特定资产协议提供的每个代币都已被借出，则Available Liquidity (S)变为 0。

将其代入公式：U = B / (0 + B) = B / B = 1。

利用率为 1（或 100%）表示池已被充分利用；所有可借出的代币目前都已被借出。

情况 2：没有借入代币（U = 0）

如果没有对代币的有效借贷，则Total Debt (B)（包括任何潜在的应计利息，如果没有借入本金，则也为零）为 0。

这Available Liquidity (S)将是一些正值，代表已提供但未借用的代币。

公式变为：U = 0 / (S + 0) = 0 / S = 0。

利用率为 0（或 0%）表示池中没有任何代币被借用；所有提供的代币均可供借用。

情况 3：部分代币被借出（0 < U < 1）

当利用率介于 0 和 1 之间时（例如，25% 为 0.25，75% 为 0.75），表示部分供应的代币正在被主动借入，同时仍有一些流动性可供新借款人使用。

利用率的主要特征和含义

了解利用率需要认识到几个重要方面：

动态价值：利用率并非静态的。随着用户借入更多代币、偿还债务，或者流动性提供者提供新资产或提取现有资产，利用率会持续实时变化。

综合债务计算：如上所述，Total Debt (B)这是一个包罗万象的数字。它包括最初借入的本金以及截至计算时该债务已产生的任何利息。这确保利用率准确反映未偿债务总额。

利率的主要驱动因素：利用率最关键的功能在于它直接影响协议内的借贷利率。虽然不同协议的具体机制可能有所不同，但有一个普遍的原则：更高的利用率通常会导致更高的利率。这种增长有助于激励更多的流动性供应（因为贷款人收益更高），同时也可以抑制借贷需求（因为借贷成本更高）。

组件之间的关系

总债务、可用流动性和利用率之间的相互作用至关重要：

总债务（B）的增加，意味着借入的代币更多，将导致利用率（U）上升（假设可用流动性S保持不变或不按比例增加）。

相反，可用流动性（S）的增加（可能是由于新存款人向池中添加代币）将导致利用率（U）降低（假设总债务B保持不变或不按比例增加）。

结论

利用率是 DeFi 借贷协议中的一个基本指标，它能够透明地衡量资产的借贷需求与其可用供应量之间的关系。通过理解利用率的计算、构成和含义，用户可以更好地理解这些协议如何管理流动性、确定利率并维持其生态系统的平衡。对于任何与 DeFi 借贷市场互动或分析的人来说，利用率都是一个基石概念。
